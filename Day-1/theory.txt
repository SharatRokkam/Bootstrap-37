What is Bootstrap ?
- Bootstrap is a powerful, feature-packed frontend toolkit. Build anything—from prototype to production—in minutes.
- is a giant collection of handy, reusable bits of code written in HTML, CSS and Javascript.
- Its also a frontend development framework that enables developer and designers to quickly build fully responsive websites
-Bootstrap is completely free to use in personal and commercial projects.

What is Responsive Design ?
- Responsive web design is the approach that suggests that design and development should respond to the user's behavior and environment based on the screen size, platform and orientation(landscape and portrait).


// Bootstrap : Container Layout 
Container are the most basic layout element in Bootstrap.  Containers are used to pad the content inside of them.
There are two container classes available.
1. .container - provides responsive fixed width container
2. .container-fluid - provides full width container, spanning the entire width of the viewport


1. Fluid Container : width is always 100%

<div class="container-fluid"></div>

2.Fixed Container : max-width :- extrasmall(xs) - 100%, small(s) - 540px, medium(m) - 720px, large(l) - 960px , extraLarge(xl) - 1140px

<div class="container"></div>  //fixed width
